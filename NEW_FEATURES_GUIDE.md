## 📋 How to Customize

### 1. Update Testimonials:
Edit `/src/app/components/Testimonials.tsx`
- Replace names, companies, photos
- Update testimonial text
- Change ratings if needed

### 2. Update Timeline:
Edit `/src/app/components/Timeline.tsx`
- Add/remove timeline items
- Update dates, descriptions
- Customize highlights

### 3. Update Resume:
Edit `/src/app/components/DownloadResume.tsx`
- Replace resume content
- Or link to a PDF file:

```typescript
const handleDownload = () => {
  // Link to your actual PDF resume
  const link = document.createElement('a');
  link.href = '/path/to/your/resume.pdf';
  link.download = 'Gobinda_Sharma_Resume.pdf';
  link.click();
};
```

### 4. Customize Theme Colors:
Edit `/src/styles/theme.css`
- Modify CSS variables for both themes
- Change gradients and colors
